# Reading List & Resources

## Foundational Concepts

### AI & Machine Learning Basics

- **[What is Machine Learning?](https://developers.google.com/machine-learning/crash-course)** - Google's ML Crash Course
  - Free, comprehensive introduction to ML concepts
  - Includes interactive exercises

- **[AI for Everyone](https://www.deeplearning.ai/courses/ai-for-everyone/)** - Andrew Ng (Coursera)
  - Non-technical overview of AI
  - Great for understanding the landscape

- **[Introduction to Deep Learning](https://www.deeplearning.ai/courses/deep-learning-specialization/)** - Andrew Ng
  - More technical deep dive
  - Includes neural networks fundamentals

### Large Language Models

- **[Attention Is All You Need](https://arxiv.org/abs/1706.03762)** - Vaswani et al.
  - The original Transformer paper
  - Foundation for modern LLMs

- **[How Large Language Models Work](https://www.3blue1brown.com/lessons/gpt)** - 3Blue1Brown
  - Visual explanation of LLM mechanics
  - Great for intuition building

- **[The Illustrated Transformer](http://jalammar.github.io/illustrated-transformer/)** - Jay Alammar
  - Step-by-step visual guide to Transformers

## Practical Guides

### Prompt Engineering

- **[Prompt Engineering for Developers](https://www.deeplearning.ai/short-courses/chatgpt-prompt-engineering-for-developers/)** - DeepLearning.AI
  - Practical techniques for better prompts
  - Uses OpenAI API examples

- **[Anthropic's Prompt Engineering Guide](https://docs.anthropic.com/claude/docs/introduction-to-prompt-design)**
  - Best practices for Claude
  - Applicable to other models too

### API Integration

- **[Google Cloud Vertex AI Documentation](https://cloud.google.com/vertex-ai/docs)**
  - Official Vertex AI guide
  - API reference and examples

- **[Gemini API Documentation](https://ai.google.dev/docs)**
  - Official Gemini API guide
  - Code samples in multiple languages

### Python for AI

- **[Python for Data Analysis](https://wesmckinney.com/book/)** - Wes McKinney
  - Pandas, NumPy fundamentals
  - Data manipulation skills

- **[Fluent Python](https://www.oreilly.com/library/view/fluent-python-2nd/9781492126249/)** - Luciano Ramalho
  - Advanced Python concepts
  - Best practices for production code

## Security & Ethics

  - Mitigation strategies

- **[Google Cloud Security Best Practices](https://cloud.google.com/docs/security)**
  - Cloud security fundamentals
  - API key management

### AI Ethics

- **[Fairness and Machine Learning](https://fairmlbook.org/)** - Barocas, Hardt, Narayanan
  - Free online book
  - Bias and fairness in AI

- **[AI Ethics Guidelines](https://www.oecd.org/going-digital/ai/principles/)** - OECD
  - International AI principles
  - Responsible AI frameworks

## Recommended Learning Path

### Week 1-2: Foundations

1. Start with "AI for Everyone"
2. Read "What is Machine Learning?"
3. Watch "The Illustrated Transformer"

### Week 3-4: Practical Skills

1. Complete Vertex AI Quickstart
2. Read Prompt Engineering guides
3. Build your first chatbot

### Week 5-6: Advanced Topics

1. Explore RAG concepts
2. Study security best practices
3. Learn about multi-agent systems

### Week 7+: Specialization

1. Choose your focus area
2. Deep dive into relevant resources
3. Build projects

## Tips for Learning

- **Learn by doing:** Build projects alongside reading
- **Join communities:** Connect with other learners
- **Stay updated:** Follow AI news and research
- **Practice consistently:** Regular coding practice is key
- **Ask questions:** Don't hesitate to reach out
- **Share knowledge:** Teaching others reinforces learning
